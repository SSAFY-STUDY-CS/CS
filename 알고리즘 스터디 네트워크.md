# 알고리즘 스터디 네트워크

- 언어: Java
- 플랫폼: 백준, 프로그래머스, SWEA
- 목표:
    - 코딩 테스트에서 자주 출제되는 알고리즘 유형을 체계적으로 학습한다.
    - 문제 조건과 시간 복잡도를 바탕으로 적절한 알고리즘을 빠르게 판단하는 능력을 기른다.
    - 코딩 테스트에서 실질적으로 도움이 되는 디버깅 방법과 엣지 케이스 분석 역량을 강화한다.
    - 대표 문제를 먼저 해결한 뒤, 다양한 접근 방식과 풀이 아이디어를 공유하고 발표한다.
    - 심화 알고리즘을 가볍게 경험하며 문제 해결의 시야를 넓힌다.
