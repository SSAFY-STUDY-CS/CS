1️⃣ 네트워크란 무엇인가

**네트워크(Network)**란

두 개 이상의 장치가 데이터를 주고받을 수 있도록 연결된 구조

예시

스마트폰 ↔ 서버 (카톡, 인스타)

노트북 ↔ 노트북 (파일 전송)

브라우저 ↔ 웹 서버 (HTTP)

📌 핵심 포인트

단순 연결 ❌

**정해진 규칙(프로토콜)**에 따라 통신 ⭕

2️⃣ 통신의 기본 요소

네트워크 통신에는 항상 아래 4가지가 필요함

요소	설명
송신자(Sender)	데이터를 보내는 쪽
수신자(Receiver)	데이터를 받는 쪽
메시지(Message)	전달하려는 데이터
프로토콜(Protocol)	통신 규칙
예시

브라우저에서 네이버 접속

송신자: 내 브라우저

수신자: 네이버 서버

메시지: “이 페이지 주세요”

프로토콜: HTTP

4️⃣ 계층화의 이유 (왜 나눴나?)

5️⃣ OSI 7계층 한 줄 요약 ⭐ (중요)
계층	이름	한 줄 설명
7	Application	사용자와 가장 가까운 계층
6	Presentation	데이터 형식/암호화
5	Session	연결 관리
4	Transport	신뢰성 & 흐름 제어
3	Network	경로 선택
2	Data Link	같은 네트워크 내 전송
1	Physical	전기 신호

👉 암기 팁

6️⃣ TCP/IP 모델 개요

OSI는 개념적 모델
실제 인터넷은 TCP/IP 모델 사용

TCP/IP	대응 OSI
Application	7,6,5
Transport	4
Internet	3
Network Access	2,1

7️⃣ OSI vs TCP/IP 차이
OSI	TCP/IP
이론적 모델	실제 사용
7계층	4계층
교육용	실무용
